# Meta Specification

## Vision

Project Chimera aims to build a scalable, autonomous influencer network where AI agents operate with economic and social agency. Agents execute campaigns, generate multimodal content, interact with audiences, and manage resources autonomously, all under ethical and safety governance.

## Constraints

* **HITL Governance**: Human review mandatory for low-confidence or sensitive content.
* **MCP Standardization**: All external interactions must go through MCP.
* **Resource Isolation**: Multi-tenant support with strict memory and wallet isolation.
* **Scalability**: System must support thousands of concurrent agents with sub-10-second latency.
* **Regulatory Compliance**: Must support AI transparency laws, including explicit disclosure of AI-generated content.
